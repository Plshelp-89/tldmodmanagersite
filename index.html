ПРИСОЕДИНИТЬСЯ

📦 КАК УСТАНОВИТЬ BEPINEX

    1. Скачайте архив (версия x64).
    2. Перетащите ВСЕ файлы из архива в корень игры, где лежит TheLongDrive.exe.
    3. Папка BepInEx должна быть рядом с .exe, не внутри других папок!
    4. Запустите игру один раз, чтобы создалась папка plugins, и закройте её.

🛠 КАК СОЗДАТЬ МОД НА C#

    1. Visual Studio с нагрузкой '.NET desktop development'.
    2. Проект: Библиотека классов (.NET Framework 4.7.2/4.8).
    3. Добавьте ссылки: Assembly-CSharp.dll, UnityEngine.dll, UnityEngine.CoreModule.dll, BepInEx.dll.
    4. Класс наследуйте от BaseUnityPlugin и поставьте [BepInPlugin].
